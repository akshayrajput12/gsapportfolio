## ğŸš¦ Getting Started

1. **Clone the repo:**
   ```bash
   git clone https://github.com/harshitvaidya/Gsap-Portfolio.git
   cd dev-web
   ```
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Run the dev server:**
   ```bash
   npm run dev
   ```
4. **Build for production:**
   ```bash
   npm run build
   ```
5. **Preview production build:**
   ```bash
   npm run preview
   ```
